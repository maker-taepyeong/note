# GitHub

소스코드를 중심으로 협업하기 위한 도구를 제공하는 서비스.

파일 시스템 기반으로 소스코드 관리, 이슈 트래킹, 코드 리뷰, CI/CD, 위키, 프로젝트 관리 등을 지원한다.

공식 문서를 읽어나가면서 적절한 워크플로우를 만들어나가야 한다.
